### Embed a Web Page

Try
```html
<iframe 
	height = 1150 
	width = 100% 
	padding = 0 0 
	margins = 0 0
	src="https://www.baidu.com/">
</iframe>
```
to embed a web page.

 - 高度只能设置为固定像素，2k像素下建议1150，1080p下建议750
 - 宽度可以设置百分比

### Underline

Try 
```html
<u>hello</u>
```
to underline your words.

### Colorful words

Try
```
<u style="color:red">hello</u>
```
to set a color to the words.
